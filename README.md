# 📝 Expense Tracker App

A full-stack Expense Tracker application built with Node.js, Express, MongoDB, JWT Authentication, and a responsive HTML/CSS/JavaScript frontend.

---

## 🚀 Features

* User Registration
* User Login with JWT Authentication
* Add Income & Expenses
* View Transaction History
* Dashboard with Balance Summary
* Delete Transactions
* Secure API with Authentication
* Responsive UI

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla JS)

### Backend

* Node.js
* Express.js
* JWT Authentication
* REST API

### Database

* MongoDB (Mongoose)
